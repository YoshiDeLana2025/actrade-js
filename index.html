Item list

Item list+

All item list

All item list+

Generates Nintendo and Nintendo Power Universal codes for items (preview limited for performance).

Batch-Search

Generate codes for a range of offsets (start offset + number of results).

Generates codes by varying byte 1 or byte 2 for a given item/player/town.
